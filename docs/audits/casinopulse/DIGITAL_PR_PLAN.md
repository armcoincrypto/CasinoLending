3. **Conference virtual attendance:** SBC, ICE — comment on live news via Twitter/X → journalist follows
4. **Podcast guesting:** Target crypto podcasts (Epicenter, Bankless adj.) and gambling podcasts
5. **HARO / Qwoted:** Respond to journalist queries on crypto payments, gambling regulation SA
